Headstart

If you are a front-end developer, you probably know how it feels like having to perform tedious tasks through numerous desktop or web applications. Optimizing images, compiling sass, concatenating scripts, etc., over and over again. Enough! Stop right now.

Enter Headstart, the easy-to-use automated front-end workflow.

Headstart is an all-in-one workflow that frees front-end developers of the little worries that come along with modern web development. If you ever wanted to use tools like Grunt or Gulp, but found the configuration too troublesome, you will probably like this better.

Get started or View this project on Github

Features

  • Command Line Interface (CLI)
  • Scaffold new projects
  • Customizable boilerplate
  • Static file serving with automated reloading
  • Simple templating
  • Image optimization
  • Sass compilation and minification
  • Javascript concatenation and uglification
  • CSS and Javascript revisioning
  • Remove unused CSS through Uncss
  • Validate your HTML through W3C
  • Hint your Javascript
  • Combine Media Queries
  • Minify HTML
  • Syncs your test setups through Browsersync
  • Tunnel static files to the web
  • Run projects through Google Pagespeed Insights
