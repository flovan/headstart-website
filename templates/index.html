Download Headstart or View the project on Github

Up and running in seconds

Headstart is built with open-source technologies and runs in your Command Line Interface (CLI). Just install the required libraries and start developing. You will be provided with a live-reload server during development to see your changes upon saving your files. Get started now!

Think fragments

Stop thinking about whether or not you included "that library you found" into your HTML. The building system keeps your styles and scripts separated untill it's finished optimizing. It also makes sure to inject everything in the right order. Just drag files into the right folders and you’re off!

No brainer optimizations

Tired of having to wield a plethora of web- and desktop applications to optimize your production files? Those days are over. The build system will take care of your code with hinting, minification, uglyfication and concatenation. Don’t forget about image and SVG optimization. Revelation? Yes!

Responsibly responsive

With the no-class grid system it becomes child's play to construct a responsive website. Keeping your media queries and grid declarations close to your selectors keeps both your HTML and CSS lean and clean.

Be the better coder

Headstart can kindly force you to follow coding conducts by hinting all of your code. Cleaner files result in better code and happier colleagues. Configurable to your heart’s content!

(Single) Page specific

By employing "view-" prefixed file names, each page will be provided with the needed assets. Nothing more, nothing less. There is even a "lite" version targeted at single page websites.

Forget about refreshing

Livereload has been around for quite a while, but you have probably been using a third party app or even a browser plugin. Headstart will automatically open a browser by choice and set you up.
